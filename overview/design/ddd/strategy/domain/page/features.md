## CRUD

데이터 관리
* Tree - 웹 페이지 보관함(폴더)

    * category
    * tag
    * description
    * parent tree id
* Shortcut - 웹 페이지 바로가기
    * title
    * category
    * tree
    * url
        * scheme
        * host(domain, port)
        * query
    * description
    * parent tree id
* Home Shortcut - 사용자 홈 화면에 표시되는 웹 페이지 바로가기
